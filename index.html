HTML Summary

HTML(Hyper Text Markup Language) is used to define our page content, structure and meaning. we don't use it for styling purposse.Use css for that insted

  • HTML USES "elements" to describe(annotate) content
  • HTML element typically have anopening tag, content and then a closing tag
  • You can also have void(empty) elements like image
  • You can also configure elements with attributes
  • Theare is a long list of available elements but we will gain experience over time, no woories

Learn more about available HTML elements on MDN HTML element referance

CSS Summary

CSS (Casscading style stylesheet) is used for styling web pages

  • To avoid code duplication , We typically use global style(eg.vai the "style" element insted)
  • Alternatively you can use external stylesheet files which we will link to
